1) Перевернуть число: 
x = 123 --> 321 
x = -456 --> -654 

2) Найти наибольший общий префикс: 
a = {"abcd", "ab6c", "acbd"} --> "a" 

3) Перевернуть пары элементов списка: 
1->2->3->4 --> 2->1->4->3 

4) Поиск пары элементов a + b = sum: 
x = [1, 6, 0, 3, -5, 2], sum = 3 --> 1 + 2, 0 + 3
